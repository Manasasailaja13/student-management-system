# Student Management System

## Project Overview

The Student Management System is a beginner-level Python project that allows users to manage student records through a menu-driven interface. It demonstrates the use of Python fundamentals such as variables, lists, loops, conditional statements, functions, and user input.

---

## Features

- View all student records
- Search a student by ID
- Add a new student
- Update student details
- Delete a student
- Exit the application

---

## Technologies Used

- Python 3
- Python IDLE

---

## Concepts Used

- Variables
- Lists
- Nested Lists
- Functions
- For Loop
- While Loop
- If-Else Statements
- User Input
- List Methods
